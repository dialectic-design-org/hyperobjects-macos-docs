# Configurations



## Pipeline

Specifies which render pipeline to use.

## Render bounding boxes

Specifies to the render pipeline that the wish is to render bounding boxes for relevant geometries.

## Free camera control

Specifies to the render pipeline that the wish is for the user to be able to control camera position.
